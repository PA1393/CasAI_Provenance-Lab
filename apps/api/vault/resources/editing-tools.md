# Guide-Design & Editing Tools

## Guide design

- CRISPR-P 2.0 (crispr.hzau.edu.cn) — plant-focused guide design and off-target scoring
- CHOPCHOP, CRISPOR — general guide design with efficiency and specificity scores
- Benchling — cloning + guide design workspace

## Scoring

- On-target efficiency and off-target specificity scores rank candidate guides
- For plants, check the reference assembly used for off-target search
