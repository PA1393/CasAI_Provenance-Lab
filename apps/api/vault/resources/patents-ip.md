# Patents & IP

## Search

- Google Patents (patents.google.com) — full-text patent search
- USPTO and Espacenet — official patent registers
- Lens.org — links patents to scholarly literature

## Editing landscape

- Foundational CRISPR-Cas9 patents (Broad Institute; UC Berkeley/Vienna) — check licensing for commercial use
- Freedom-to-operate review is separate from regulatory approval
