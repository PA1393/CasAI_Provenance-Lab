# Genome & Sequence Databases

## Core databases

- NCBI Nucleotide / GenBank — sequences and accessions
- Ensembl Plants — plant genome browser, assemblies, and comparative genomics
- Phytozome (JGI) — plant reference genomes and gene families

## Use in editing

- Confirm the exact reference assembly version before guide design (coordinates shift between assemblies)
- Pull the target locus sequence and flanking region for PAM scanning
