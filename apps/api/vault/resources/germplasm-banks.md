# Germplasm Banks

## Collections

- USDA GRIN (ars-grin.gov) — US national plant germplasm system
- CIMMYT — maize and wheat genetic resources
- IRRI Genebank — rice accessions

## Use in editing

- Source accessions and passport data for the genetic background being edited
- Record the exact accession ID for [[topics/field-trials|reproducibility and regulatory filings]]
