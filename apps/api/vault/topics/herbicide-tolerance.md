# Herbicide tolerance

## Common gene targets

- EPSPS — glyphosate tolerance; specific codon changes reduce herbicide binding while preserving enzyme function
- ALS / AHAS — sulfonylurea and imidazolinone tolerance via point mutations (e.g. P197, W574)

## Editing approach

- Base editing is well suited: a single C→T or A→G recreates a known tolerant allele without inserting a transgene
- Guide must place the target codon inside the editing window and keep other editable bases out of it
