# Field trials

## Regulatory gate

- Confined field trials often require permits or notifications (see [[regulators/usda-aphis|USDA APHIS]])
- Product-class / regulatory-status determination precedes any unconfined release

## Records to keep

- Genetic background / accession ID, edit description, guide RNA, and the exact reference assembly used
- Provenance of each result so a run can be reproduced and audited
