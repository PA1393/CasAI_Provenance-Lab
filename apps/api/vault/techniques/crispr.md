# CRISPR-Cas9 in crops

## Mechanism

- An sgRNA-guided SpCas9 cuts ~3 bp upstream of an NGG PAM, creating a double-strand break
- NHEJ repair yields small indels (gene knockouts); HDR with a donor template enables precise edits but is inefficient in plants

## Delivery in plants

- Agrobacterium-mediated transformation; biolistics; protoplast/PEG for transient validation
- Regeneration from callus is the practical bottleneck for many crops
