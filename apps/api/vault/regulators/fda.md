# FDA — Agricultural Biotechnology

**Site:** https://www.fda.gov/food/agricultural-biotechnology

## What to track

- Plant Biotechnology Consultation program (voluntary premarket food-safety consultation)
- Focus on food/feed safety: allergenicity, toxicity, nutritional composition
- Coordinates with [[regulators/usda-aphis|USDA]] and [[regulators/epa|EPA]] under the Coordinated Framework
